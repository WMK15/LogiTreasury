# ArcLogistics Treasury - System Architecture

## Overview

ArcLogistics Treasury is a unified logistics treasury and payout engine for European logistics companies. The system ensures **no capital sits idle** through automated yield optimization, instant FX conversion, cross-chain settlement, and seamless fiat integration.

## Track Alignment

### Track 1: Global Payouts & Treasury
- **Circle Wallets**: Operator wallet management
- **Circle Gateway**: Unified cross-chain balance abstraction
- **Arc Bridge Kit**: Cross-chain settlement routing
- **CPN**: Fiat on/off ramp integration

### Track 2: USYC & StableFX
- **USYC**: Yield layer for idle capital
- **StableFX**: FX conversion (USDC вҶ” EURC)

---

## Key Design Principles

1. **No Idle Capital**: All funds earn yield via USYC when not in use
2. **Unified Balance**: Single view across all chains via Circle Gateway
3. **Atomic Operations**: All FX and settlements are atomic (PvP)
4. **Event-Driven**: All state changes emit events for indexing
5. **Role-Based Access**: Owner вҶ’ Admin вҶ’ Operator hierarchy
6. **Fail-Safe**: Refund mechanisms for failed operations
7. **Compliance-Ready**: KYC/AML hooks for CPN integration
